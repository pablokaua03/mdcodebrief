## O que é?

Ferramenta desktop leve que varre recursivamente um projeto e gera um único Markdown com:
- Árvore visual de diretórios
- Código completo com syntax highlighting
- Caminhos, tamanhos e truncagem automática

---

## Início rápido

```bash
python mdcodebrief.py
```

Ou baixe o `.exe` e execute direto — sem instalar Python.
